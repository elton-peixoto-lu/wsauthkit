# Code of Conduct

## Our standard

This project is committed to a respectful, professional, and welcoming community.

Examples of expected behavior:

- being constructive and respectful
- assuming good intent
- giving actionable technical feedback
- keeping discussion focused on the work

Examples of unacceptable behavior:

- harassment or personal attacks
- discriminatory language
- bad-faith disruption
- hostile or demeaning communication

## Enforcement

Project maintainers may remove, edit, or reject contributions and interactions that do not align with this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the project maintainer.
