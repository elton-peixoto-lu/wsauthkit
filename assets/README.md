# Assets

This directory contains the open-source branding kit for `WSAuthKit`.

## Structure

```text
assets/
├── logo/
│   ├── logo-dark.png
│   ├── logo-light.png
│   ├── logo.svg
│   ├── monogram.png
│   ├── symbol-dark.png
│   └── symbol-light.png
├── favicon/
│   ├── favicon-16.png
│   ├── favicon-32.png
│   └── favicon.ico
└── social/
    └── github-social-preview.png
```

## Usage

- `logo-light.png`: README and docs on light backgrounds
- `logo-dark.png`: dark UI and GitHub dark theme screenshots
- `logo.svg`: scalable documentation and website usage
- `symbol-light.png`: compact avatar on light backgrounds
- `symbol-dark.png`: compact avatar on dark backgrounds
- `monogram.png`: minimal `WS` mark
- `favicon/*`: browser and docs favicon variants
- `social/github-social-preview.png`: GitHub social preview / Open Graph image

## Design notes

The identity is intentionally:

- infrastructure-oriented
- security-focused
- Go-inspired without a playful mascot treatment
- minimal enough for open-source docs, package listings and cloud tooling sites

Core visual elements:

- a stylized Go-inspired face
- shield and padlock security motif
- bidirectional WebSocket signal waves
- blue and navy palette with neutral backgrounds

## Regeneration

Brand assets are generated by [`scripts/render_brand_assets.py`](../scripts/render_brand_assets.py).
